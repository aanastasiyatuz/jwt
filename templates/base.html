JWT Auth API

URLS:

  • 'register/' - register new user

    POST request { "username":"some_username", "password":"some_password" }

  • 'api/token/' - authenticate a user (returns access and refresh token)

    POST request { "username":"some_username", "password":"some_password" }

  • 'api/token/refresh/' - refresh expired token (access (10 minutes), refresh (1 day))

    POST request { "refresh":"refresh_token", }

  • 'check-token/' - check you are using token correctly

    POST request headers: { "Authorization": "Bearer access_token" }
